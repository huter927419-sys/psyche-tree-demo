# 实现层 · 实践篇

[01–06 世界观](../01-本源.md) 立「人如何存在于世」；**本目录立「若此真，体验应如何践行」**。

不是接口说明文档，不是功能清单。正文答：**理论要求何种对待人的方式**；「实现映射」说明理论为何落成如此践行，不作代码目录。

**读序：** [附录 · U](../appendix-现代对应.md) → [01–06 世界观](../01-本源.md) → **本目录 01 → 08**。修持文案 [volume-rite-copy.md](../../volume-rite-copy.md) 不改；实践篇解释**为何**须慢、须照、须留痕、须归树——而非复述引导语。

各章 **简体 · English · 日本語** 并列：开篇「实践问 / 理论根源」有段外译文；各节 blockquote 内三节对照。

## 章体模板（01 → 08 统一）

六卷体验、纸本、工作坊共用同一骨架——**七节、三节名一致**：

| 层 | 节 | 问什么 |
|----|-----|--------|
| **哲学** | 实践问 · 理论根源 · 章眼 · **理论原则** | 为什么要这样对待人（**为何**） |
| **实践** | **实践原则** · **不可与可** | 体验应如何发生（**如何体现** + 边界） |
| **实现** | 实现映射 · 理论位置 · 本章收 | 原则如何落到读卷节点 |

```
实践问
  ↓
理论根源
  ↓
一 · 章眼
  ↓
二 · 理论原则（为何；N 则，无 → 映射）
  ↓
三 · 实践原则（如何体现；→ 对应 + 仪轨链，不单立「仪轨」节）
  ↓
四 · 不可与可（原「戒律」；不可 / 可，非戒 / 行）
  ↓
五 · 实现映射（实践映照）
  ↓
六 · 理论位置
  ↓
七 · 本章收（08 为「本章收 · 实践层收」）
```

**说明：**

- [01 存在论](./01-存在论.md) 起，[02](./02-意识论.md)–[08](./08-整合论.md) **同构**。
- **理论原则 / 实践原则 / 不可与可** 三节名全章一致；各章仪轨链**并入第三节**，可增删链节，不单独立节。
- 01 理论原则为三**则**；02–08 多为五**则**（08 理论节另含「涌现 / 整象不归终」收束句）。

## 一次旅程中的践行序

```
书架择卷 → 入卷（觉） → 问印（照·应·留形） → 离卷（息）
    → 六卷各留一息 → 归树（向·收束） → 整象（齐观·生成）
```

| 节点 | 践行要义 | 主章（节） |
|------|----------|------------|
| 入卷 / 注意力 | 从惯性应答中拉回，开叉入口 | [02](./02-意识论.md) 理论原则 · 实践原则 |
| 问印择印 | 此息接缘留形，非考试 | [05](./05-因应论.md) · [08](./08-整合论.md) 实现映射 |
| 卷内结果 / 鸣示 | 此息之象，非终身型；从痕显现 | [01](./01-存在论.md) · [06](./06-生成论.md) |
| 同卷再入 | 字如故，读者已变 | [03](./03-流动论.md) 不可与可 · 今日不否昨日 |
| 六卷同旅 | 六痕互文，非六块报告 | [04](./04-关系论.md) 实践原则 |
| 归树 → 整象 | 树未变，看树者变；齐后开口 | [07](./07-意义论.md) · [08](./08-整合论.md) |

## 改体验前 · 四问

1. 是否在**贴型**或**判对错**？→ [01 存在论](./01-存在论.md) **不可与可** · [05 因应论](./05-因应论.md) **不可与可**  
2. 是否在**催促改**而跳过照见？→ [02 意识论](./02-意识论.md) **理论原则** · **不可与可**  
3. 是否把结果**焊死**或**拼接**？→ [03 流动论](./03-流动论.md) · [08 整合论](./08-整合论.md) **不可与可**  
4. 是否在**绑人**或**替人应**？→ [04 关系论](./04-关系论.md) **不可与可**

**改读卷设计前（整合论 · 实现映射）：** 是否拆维？是否平均？是否拼接？是否单卷代齐观？
