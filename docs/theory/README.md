## 世界观 · 章体

各篇结构一致，与 [implement](./implement/README.md) 的七节模板**不同**（那是践行层专用）：

```
承上篇 · 本篇核心问（段外亦有三语）
    ↓
第一章 … （02 观照 第二章 **息间**，七章含动力学）
    ↓
本篇偈 · Gatha · 本篇の偈（四行收韵，三语并列）
    ↓
本篇收 · 本篇、収め（章眼 / 一句 / 下篇指向）
    ↓
→ 承上 · 下篇 · （部分篇链 implement / 附录）
```

- 正文以 **blockquote 三语** 展开论证，不作接口说明。
- **本篇偈** 于各篇末、本篇收之前：四行偈语，**简体 · English · 日本語** 并列，收摄本篇要义。
- **本篇收** 常含 **章眼** 一句（03 流动、04 因应、05 共生 等）；01 本源以三问表收束；02 观照以「观照之要 / 动力学之要」收束。
- 六篇读序固定；**实践八章**按践行依赖排列，与世界观编号略有交错（见下表「实践对应」）。
