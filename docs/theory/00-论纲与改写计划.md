# 雾岸理论 · 论纲与改写计划

> **状态（2026-07）：** 已按本论纲完成 **01–10 正文 + 附录**；旧 `01–05` 在 [archive/](./archive/)。

> 读完旧五篇后的共识：**问题不在某一句写得不好，而在每一章都在重新定义世界。**  
> 改写的核心原则：**雾岸是第一语言；心理学、动力学、State/Field/Flow 是第二语言，放附录。**

---

## 一、诊断（对照现有文件）

### 01 玄学体系 — 方向对，后半越界

**保留（9.5 分）：** 雾岸、光柱、六卷、树、留印、返岸、整象为何最后出现、冥想/祈祷/反思作为**体验结构**。

**应移出本章：**

- 整节「二、场域、能量与心流」（现 `01` §二）— 定义 Field / Energy / Flow，属于后期理论，不是世界观。
- 开篇与 §一 末尾反复出现的「在场域中安住 → 察能量 → 心流择印」— 改为**现象描述**（「慢下来、择印、合卷」），不出现术语。
- 体系总览表里「场域·能量·心流」一行 — 改为「修持环 / 六向 / 神示」等**读者能感到**的层，不提前命名理论。

**第一章只回答一个问题：我来到哪里？**  
只负责让读者**相信这里存在**，不负责解释机制。

---

### 02 简明理论 — 断裂最大

**现状：** 第一章还在树、雾、光；第二章直接 `U = [认知, 情绪, …]`，雾岸世界消失。

**应改为：** 不叫「简明理论」，建议 **照见论** 或 **心象论**。

**只回答：** 为什么六卷能照见人？为什么每人看见的雾不同？为什么六卷各照一面？为什么整象不能提前？

**建议顺序（正文）：**

1. 什么是照见？（不是测评、不是打分）
2. 为什么不同的人看见不同的雾？
3. 为什么六卷只能照见一个侧面？
4. 为什么整象不能提前出现？
5. **最后**才说：现代心理学把上述结构读成认知、情绪、关系…… — **解释层，不是主体**

现有 `02` 里「六维向量、演化方程、命运模型」等**保留素材**，但降级到附录或各章末尾「现代对应」。

---

### 03 进阶泛化 — AI 味最重

**现状：** State Object / Force Object / Field Object 打头，像软件设计文档。

**应改为：** **万象论** 或 **诸象论**。

**只回答：** 为什么雾岸不仅能照见人，还能照见万物？

**正文顺序：** 树、河流、国家、组织、AI…… 都是「象」— 同一照见逻辑的不同载体。  
**最后：** 现代语言里，这种「象」有时写成 State Object — **一句对应即可，正文不用 Object 小标题。**

---

### 04 增强理论 — 内容好，顺序错

**现状：** Flow · Awareness · Field 三节并列 + 大量 U Φ A F 公式。

**应拆职责：**

- **本章只讲心流（流）：** 为什么同一个人有时会进入非常顺畅的状态？（与产品「流衡卷 / 一页一印」接轨）
- **意识（观）** 单独成篇或紧接下一章：是谁知道「我进入了心流」？
- **场域（场）** 与修持意象接轨：为什么站在湖边、为什么丝线、为什么星空 — 在修持卷里**感受**，在理论里**命名**

**U、Φ、A、F 及统一公式 → 全部移附录 A《现代符号对应》。**

---

### 05 形而上学扩展 — 跨度太大

**现状：** 意识流 → 现实 → 宇宙 → 本源 → 因果 挤在一章。

**应拆成三个问题（可三文件，或一章三节严格递进）：**

| 节 | 问题 | 禁止 |
|----|------|------|
| 1 意识 | 什么叫意识？ | 不要先谈宇宙 |
| 2 现实 | 意识如何形成**体验**？ | 不要直接断言「现实是意识生成」— 先推导 |
| 3 因果 / 命 | 路径为何固定？为何像命运？ | 宇宙、本源放**最后** |

现有 `05` §四「宇宙本源」应后移，接在因果推导之后。

---

### 修持文案（volume-rite）— 灵魂在，缺桥

**现状：** 湖 / 河 / 星空 / 丝线 — 极好，像一本书；与理论术语无显式对应。

**目标：** 理论说**是什么**，修持说**怎么做**。

| 理论（后文） | 修持（已有意象） |
|--------------|------------------|
| 观照 | 入卷静坐、允许沉默 |
| 心流 | 一页一印、不评判 |
| 场域 | 湖岸、河面、夜空 |
| 缘 | 丝线 |
| 归树 / 整象 | 归树仪式 → 书架整象 |

改写时在 `01` / 照见论 / 修持全文之间加**交叉引用**即可，不必改产品 UI。

## 二、目标结构（九篇 + 附录）

读者路径：**先活在雾岸世界里，最后才「发现」和现代科学的对应。**

| 篇 | 工作标题 | 只回答一个问题 | 与六卷 / 产品 |
|----|----------|----------------|---------------|
| 1 | 我来到哪里？ | 雾岸是什么、为何在此 | 书架、树、雾、留印 |
| 2 | 心象论 | 为何会照见自己 | 六卷总义 |
| 3 | （照见论续） | 为何每人雾不同 | 同一问印不同择印 |
| 4 | 生长 | 人为何会变 | 树脉亮起、单卷完成 |
| 5 | 缘 | 人如何相互影响 | 缘书卷 |
| 6 | 流 | 为何有顺畅状态 | 流衡卷、问印节奏 |
| 7 | 观 | 谁知道自己正在变 | 注意力页、归树 |
| 8 | 命 | 轨迹为何固定 | 形而上学·因果（前半） |
| 9 | 整 | 整象为何只能最后 | 归树 → 整象 |

> 说明：上表 2–3 可合并为现在的「照见论」一篇；8–9 可合并为「形而上学」一篇。九是**职责**划分，不必机械拆九个文件。

**附录（第二语言）：**

- 附录 A：符号（U, Φ, A, F, C(t), Ω …）
- 附录 B：与心理学 / 动力学 / 复杂系统的对照表
- 附录 C：旧版五篇全文归档（改写完成前保留 `01–05` 只读）

## 五、一句话

**不应从 State、Flow 推导出雾岸；而应让读者在雾岸里走完九问，然后在附录里恍然大悟：「原来 psychologists 也这么叫。」**

现有五篇是**素材库**；本论纲是**职责表**。按职责重写后，AI 味会主要来自附录和工程文档，而不是哲学正文。
